Suspense


If suspense was a color,

It would be neon green

As bright as a highlighter.

If suspense was a taste

It would be just like sour patch kids.

If suspense was a feeling

It would be as prickly as a cactus.

If suspense was a smell

It would be sour as a lemon.

If suspense was a sound,

It would be as dramatic as a drum roll.


Hope


If hope could be a color

It would be rainbow,

As colorful as fireworks.


If hope could be a taste,

It would taste just like piping hot cookies.


If hope could be a smell

It would be Sharpies.


If hope could be a sound

It would be a drum solo.


If hope could be a feeling

It would be soft.


If hope could be an animal

It would be a mouse.
